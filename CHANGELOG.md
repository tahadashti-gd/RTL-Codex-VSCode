# Changelog

All notable changes to this project are documented in this file. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project uses [Semantic Versioning](https://semver.org/).

## [Unreleased]

## [0.2.0] - 2026-07-19

### Added

- English command names and user-facing messages
- A clearer public title identifying the project as a VS Code extension
- English and Persian documentation
- GitHub Actions CI and community health files
- Arabic, Latin, and Latin Extended Vazirmatn subsets
- Integration coverage for the patch and restore lifecycle

### Fixed

- Invalid characters in the generated input stylesheet

## [0.1.0] - 2026-07-19

### Added

- Initial reversible RTL patcher
- Vazirmatn variable font support
- Enable, disable, and status commands
